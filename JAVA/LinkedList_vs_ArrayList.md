# ArrayList 와 LinkedList 비교

| 컬렉션     | 읽기(접근시간) | 추가/삭제 | 비고                                             |
|------------|----------------|-----------|--------------------------------------------------|
| ArrayList  | 빠르다         | 느리다    | 순차적인 추가 삭제는 빠름 비효율적인 메모리 사용 |
| LinkedList | 느리다         | 빠르다    | 데이터가 많을수록 접근성이 떨어짐                |
